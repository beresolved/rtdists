rtdists: Response time distributions in R
====

## Features

* uses fast-dm C code by Andreas Voss

## Installation

* From CRAN: In the future

* Development version from Github:
```
devtools::install_github("rtdists",user="rtdists")
```

## Get Started:
```
rrd(10, c(rep(1, 4), rep(0.1, 4)))
```
